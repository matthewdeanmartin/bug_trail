# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.0.0] - 2025-06-21
### Changed
- Remove picologging and ctags dependencies
- Code modernization and formatting

### Fixed
- Build script improvements

## [2.3.0] - 2024-01-10
### Added
- File watch support
- Data enrichments

## [2.0.1] - 2024-01-05
### Changed
- Fix `--config` CLI argument to accept a string path rather than a boolean flag
- Add `--version` output that reflects the actual installed package version
- Enable verbose logging via `--verbose` flag using standard logging configuration

## [2.0.0] - 2024-01-05
### Changed
- 30% less code ugliness

## [1.0.1] - 2024-01-04
### Changed
- Add quality-of-life debug logging to data fetching, filesystem utilities, and view rendering
- Fix version number reporting in the installed package metadata

## [1.0.0] - 2024-01-04
### Added
- Initial release
- Performance and multithreading support
- Split into separate libraries
- Quality of life logging improvements
